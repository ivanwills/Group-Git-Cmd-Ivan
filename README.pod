=head1 Group-Git-Cmd-Ivan

My own command additions to the standard group-git commands.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install

=head1 SUPPORT AND DOCUMENTATION
